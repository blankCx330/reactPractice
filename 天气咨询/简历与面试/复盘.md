### 1. buffer 为什么会重复输出？

```typescript
// 正常流程
buffer = 'data: {"content":"今' // chunk 1 不完整
lines = ['data: {"content":"今']
buffer = 'data: {"content":"今' // pop() 返回最后一个（不完整的）

// chunk 2 到达
buffer = 'data: {"content":"今天"}\n\ndata: ...'
lines = ['data: {"content":"今天"}', 'data: ...']
buffer = '' // pop() 返回空字符串

// ❌ 如果删除 buffer = lines.pop()
buffer = 'data: {"content":"今天"}\n\ndata: ...' // 没有重置！
// 下次循环又包含同样的数据，重复解析
```

## 三、知识点的层级理解

### 3.1 第一层：知道"是什么"

- SSE 是服务器单向推送技术
- ReadableStream 是可读流
- AbortController 可以取消请求

**这层只能让你"认识"代码，不能让你"写"代码。**

---

### 3.2 第二层：知道"为什么"

| 知识点          | 为什么需要它？                                                 |
| --------------- | -------------------------------------------------------------- |
| SSE             | AI 生成内容是逐步产生的，用 SSE 可以实时显示，不用等全部生成完 |
| ReadableStream  | HTTP 响应的 body 默认是一次性返回的，流式响应需要逐步读取      |
| AbortController | 用户可能中途离开，不取消会浪费资源，还可能导致内存泄漏         |
| buffer          | 一个数据块可能包含不完整的一行，需要暂存等下一个块拼接         |
| 函数式更新      | 流式场景下状态更新很频繁，闭包可能捕获旧值                     |

---

### 3.3 第三层：知道"什么时候用/不用"

| 场景                   | 用 SSE？用 WebSocket？用普通 fetch？ | 评判        | 解析                                                                                                                                                                      |
| ---------------------- | ------------------------------------ | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 用户登录验证           | webSocket                            | ❌ **错误** | 登录验证是**一次性请求-响应**操作，只需发送账号密码，服务器返回验证结果。用 WebSocket 大材小用，且增加复杂度（需要握手升级协议、保持长连接）。<br><br>**正确答案：fetch** |

```typescript
function parseSSE(raw: string): string[] {
  const result: string[] = []
  const lines = raw.split('\n')

  for (const line of lines) {
    if (line.startsWith('data: ')) {
      const data = line.slice(6).trim()
      if (data === '[DONE]') continue
      try {
        const json = JSON.parse(data)
        const content = json.choices?.[0]?.delta?.content
        if (content) result.push(content)
      } catch {}
    }
  }

  return result
}
```

function correctParse(chunks: string[]): string[] {
  const result: string[] = []
  let buffer = ''

  for (const chunk of chunks) {
    buffer += chunk
    const lines = buffer.split('\n')
    buffer = lines.pop() || ''

    for (const line of lines) {
      if (line.startsWith('data: ')) {
        const data = line.slice(6).trim()
        try {
          const json = JSON.parse(data)
          const content = json.choices?.[0]?.delta?.content
          if (content) result.push(content)
        } catch {}
      }
    }
  }

  return result
}
